The Basics

What is an HTML file?

HTML is just a text file with a .html extension. Save it to your desktop, double-click it — and it opens instantly in any browser, with no internet required.

No app to install. No server to set up. No login. Just a file.

Inside that file, you describe what to show — text, charts, buttons, forms — and the browser handles the rest.

And today, you don't need to write any of it yourself.

my_tool.html 
<!-- A complete, runnable file -->
<html>
<head>
  <title>My Report</title>
</head>
<body>
  <h1>Q2 Sales Dashboard</h1>
  <canvas id="chart"></canvas>

  <script>
    // Charts, tables, logic …
    const data = [42, 58, 71, 90];
    // AI generated all of this
  </script>
</body>
</html>

Publishing to the Web — the Easiest Way

GitHub Pages: Free Hosting in 5 Steps

Your local HTML file becomes a live URL anyone can open — no server, no cost, no backend. Just GitHub.

STEP 01
🐙

Create a GitHub account

Free at github.com — takes 2 minutes

STEP 02
📁

Create a repository

Click New → name it, set to Public

STEP 03
⬆️

Upload your file

Drag & drop your .html file. Name it index.html

STEP 04
⚙️

Enable Pages

Settings → Pages → Branch: main → Save

STEP 05
🎉

Your site is live

GitHub gives you a public URL in ~60 seconds
